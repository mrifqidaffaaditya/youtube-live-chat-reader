YouTube Live Chat Reader

Selamat datang! Alat ini akan membaca chat dari live stream YouTube Anda dan mengubahnya menjadi suara (TTS) secara otomatis.

Cara Penggunaan:

  • Masukkan URL Live YouTube atau Video ID di panel atas.
  • Klik Mulai Mendengarkan. Chat akan mulai bermunculan dan suara akan diputar.

Integrasi OBS:

  • Tambahkan Browser Source di OBS dan masukkan URL: http://localhost:25522/?obs=1
  • Parameter ?obs=1 akan menyembunyikan panel kontrol dan tutorial agar background menjadi transparan.

Dukungan Lavalink:

  • Alat ini juga menyediakan endpoint audio untuk bot musik/Lavalink pada: /audio/:id.mp3
