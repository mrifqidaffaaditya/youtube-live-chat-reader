YouTube Live Chat Reader

Selamat datang! Alat ini akan membaca chat dari live stream YouTube Anda dan mengubahnya menjadi suara (TTS).

Cara Penggunaan (Browser Biasa):

  • Masukkan URL Live YouTube atau Video ID di panel atas.
  • Klik Mulai Mendengarkan.

Cara Penggunaan (OBS Overlay):

  • Tambahkan Browser Source di OBS dan masukkan URL dengan format berikut:
    http://localhost:25522/?obs=1&v=ID_VIDEO_YOUTUBE
  • Ubah ID_VIDEO_YOUTUBE dengan ID Video atau URL YouTube yang sedang live.
  • Parameter ?obs=1 berfungsi menyembunyikan panel atas dan kotak tutorial ini agar background menjadi sepenuhnya transparan.

Dukungan Lavalink:

  • Tersedia endpoint audio di: /audio/:id.mp3
